# Bankmarketing_prediction
The Bank Marketing Dataset from Kaggle contains real data from direct marketing campaigns conducted by a Portuguese bank.
We have developed a machine learning model that accurately predicts whether a customer will subscribe to a term deposit based on their demographic, financial, and campaign-related information.
This data includes numerical and categorical features such as:
Customer information: age, job, marital status, education
Financial details: bank balance, loan statuses
Campaign information: number of contacts, previous campaign outcome, call duration

To accurately predict whether a customer will subscribe to a term deposit, we implemented and compared four different machine learning algorithms, as follows:
1. Logistic Regression: is a supervised learning algorithm used for binary classification problems. It predicts the probability of an outcome (Yes/No) using a mathematical function.

2. Decision Tree: is a tree-based model that makes decisions by splitting data into branches based on feature conditions. It is easy to understand and interpret.

3. Random Forest: is an ensemble learning algorithm that combines multiple decision trees to improve accuracy and reduce overfitting. It gives more stable and reliable predictions compared to a single decision tree.

4. XGBoost (Extreme Gradient Boosting): is an advanced ensemble algorithm that builds trees sequentially, where each new tree corrects the errors of the previous one. It is powerful, efficient, and often provides high accuracy in classification problems.
